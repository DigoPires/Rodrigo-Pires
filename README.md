# 🚀 Portfólio Profissional - Rodrigo Pires Figueiredo

Uma landing page profissional e moderna desenvolvida para apresentar o currículo e portfólio de Rodrigo Pires Figueiredo, Analista de Dados e Desenvolvedor Fullstack.

## 📋 Visão Geral

Este portfólio profissional foi criado para destacar as habilidades profissionais, experiência acadêmica e competências técnicas de Rodrigo Pires. O design é moderno, totalmente responsivo e inclui animações suaves para proporcionar uma excelente experiência ao usuário, além de funcionalidades avançadas como modo claro/escuro e seção de serviços.

## ✨ Características

- 🎨 **Design Moderno**: Interface limpa e profissional com gradientes e sombras sutis
- 📱 **Totalmente Responsivo**: Adaptado para desktop, tablet e dispositivos móveis
- ⚡ **Animações Suaves**: Efeitos de scroll, hover e transições interativas
- 🌓 **Modo Claro/Escuro**: Alternância de tema com preferência do usuário
- 🔗 **Integração Social**: Links diretos para LinkedIn, GitHub, WhatsApp e Instagram
- 📊 **Visualização de Habilidades**: Tags de habilidades organizadas por categoria
- 🚀 **Seção de Serviços**: Cards animados para oferta de serviços profissionais
- 🎯 **SEO Otimizado**: Semântica HTML5 apropriada e favicons
- 📍 **Logo Fixa**: Logo pessoal no canto superior esquerdo
- ⬆️ **Botão Voltar ao Topo**: Navegação aprimorada

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Design moderno com Grid, Flexbox, variáveis CSS e animações
- **JavaScript Vanilla**: Interatividade, animações e alternância de tema
- **Font Awesome 6.4.0**: Ícones profissionais e modernos
- **Google Fonts**: Tipografia Inter para melhor legibilidade

## 🚀 Como Usar

### 1. Visualizar a Página

1. Abra o arquivo `index.html` em seu navegador preferido
2. Ou use um servidor local para melhor desenvolvimento:
   ```bash
   # Usando Python
   python -m http.server 8000
   
   # Usando Node.js
   npx serve .
   
   # Usando VS Code Live Server
   ```

### 2. Personalizar Informações

As informações estão organizadas nos seguintes arquivos:

- **`index.html`**: Estrutura e conteúdo principal
- **`styles.css`**: Cores, fontes, layout e animações
- **`script.js`**: Animações, interatividade e funcionalidades

## 🎨 Personalização

### Cores

As cores principais estão definidas no arquivo `styles.css` com variáveis CSS:

```css
:root {
    --primary-color: #8b5cf6;    /* Roxo principal */
    --secondary-color: #7c3aed;  /* Roxo escuro */
    --accent-color: #a78bfa;     /* Roxo claro */
    /* ... outras variáveis */
}
```

### Modo Claro/Escuro

O tema é automaticamente detectado baseado na preferência do sistema, mas pode ser alternado manualmente através do botão no canto superior direito.

### Logo e Favicons

- **Logo**: Adicionada no canto superior esquerdo (`img/rp2.png`)
- **Favicons**: Configurados para diferentes tamanhos e navegadores

## 🚀 Implantação

### GitHub Pages

1. Faça upload dos arquivos para um repositório GitHub
2. Vá para Settings > Pages
3. Selecione a branch `main` e pasta `/root`
4. A página estará disponível em `https://username.github.io/repository`

### Netlify

1. Conecte seu repositório GitHub ao Netlify
2. Configure as build settings (não necessário para site estático)
3. Implante automaticamente

### Vercel

1. Importe seu repositório GitHub
2. Configure as configurações do projeto
3. Implante com um clique
